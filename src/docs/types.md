# Types

## `VerificationData`

```ts
interface VerificationData<T = any> {
  k: string; // Key/id
  p: string; // public key
  n: number; // nonce
  h: string; // hash of JSON.stringify(value) + nonce
  t: number; // Timestamp this was created
  s: string; // signature
  v: T; // value
}
```
